DatabaseConsole sovellus

Tässä projektissa päätehtävänä oli luoda konsoliohjelma, joka ottaa yhteyden SQL-tietokantaan c#:lla. Konsoliohjelma antaa käyttäjälle mahdollisuuden valita kahdesta konsoliikkunassa näkyvästä vaihtoehdosta (Asiakkaat (1) tai Toimitusyritys (2)). Olen muodostanut SQL-yhteyden käyttämällä SQL Server Instance -merkkijonoa: (Palvelin = myServerName \ myInstanceName; Database = myDataBase; User Id = my Username; Password = myPassword;) sitten käytän if-lausetta, joten ohjelma ymmärtää käyttäjän antaman vastauksen ja lähettää sen konsolille käyttäjän tekemän pyyntöikkunan. Yllä olevat kuvat näyttävät, mitä käyttäjä saa tietyllä valinnalla. Lisäksi loin yksinkertaisen SQL-tietokannan ensisijaisella avaimella ja viiteavaimella.
